# Box

Manage your paperwork the smart way, digitally.

## Available commands

```bash
$ box -h                                                                                                                                                                                                                                       ✹ ✭
usage: box [-h] {add,rm,mod,sql,find,init,show} ...

positional arguments:
  {add,rm,mod,sql,find,init,show}
                        commands
    add                 add a new file
    rm                  rm a file
    mod                 modify a file
    sql                 type custom sql
    find                find files
    init                initial db setup
    show                show file info

optional arguments:
  -h, --help            show this help message and exit

```
